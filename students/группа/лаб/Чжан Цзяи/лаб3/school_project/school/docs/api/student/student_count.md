# Показать количество студента

**URL** : `students/count/<int:pk>`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{[]}`
```json
{
    "Общие студентки": 1,
    "Общие студенты": 2
}
```
