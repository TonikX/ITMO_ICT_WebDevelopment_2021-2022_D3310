# Показать одну аудиторию

**URL** : `classrooms/<int:pk>`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{[]}`

```json
{
    "id": 1,
    "room": "b",
    "number": "1000"
}
```
