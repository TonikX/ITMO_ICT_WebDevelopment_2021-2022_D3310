# Показать количество аудитории

Выводит информацию обо всех войнах

**URL** : `classrooms/count`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{[]}`
```json
{
    "Общие  базовые": 4,
    "Общие профильные": 4
}
```
