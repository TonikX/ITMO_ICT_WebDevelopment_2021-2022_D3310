# Создайте новую книгу

Создайте новую модель книги в базе данных с помощью api

**URL** : `/books/create/`

**Method** : `POST`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : 

```json
{
    "Success": "Book '9981' created succesfully."
}
```
